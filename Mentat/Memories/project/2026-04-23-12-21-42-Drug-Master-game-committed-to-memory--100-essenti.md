# Memory

[[Drug Master]] game committed to memory — 100 essential hospital drugs organized by category (cardiovascular, antibiotics, analgesics, respiratory, endocrine, GI, psychiatry, miscellaneous) with three learning modes: flashcard (flip-card with progress tracking), quiz (multiple choice with scoring), and match game. Designed as interactive study tool for prescribing exam prep and final year medical school. Jackson can pull it anytime to practice dosing, indications, and drug names.
