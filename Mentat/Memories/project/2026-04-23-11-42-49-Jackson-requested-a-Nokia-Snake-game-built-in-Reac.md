# Memory

Jackson requested a Nokia Snake game built in [[React]] with Nokia 3310 aesthetic. [[Mentat]] delivered a fully functional game with green phosphor screen, D-pad controls, score tracking, progressive speed increase, start/retry screens, collision detection, food spawning, and high score tracking. Game is playable via arrow keys or on-screen D-pad buttons.
