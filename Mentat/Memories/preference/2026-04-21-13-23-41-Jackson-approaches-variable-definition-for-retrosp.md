# Memory

Jackson approaches variable definition for retrospective studies by first auditing the actual database — checking what's recorded and how consistently — rather than prespecifying variables in a vacuum. He recognizes this foundational step is critical and that retrospective studies often fail when analysis is planned before verifying data actually exists in usable form. He prioritizes checking patchy variables like smoking status and race early, then lets variable definitions follow from what the data can realistically support.
