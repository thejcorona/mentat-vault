# Memory

Jackson is interrogating his Lumra cEDH decklist card-by-card with [[Mentat]]. Key cards under review for intended role/synergy: Ba Sing Se (land ramp/theft tech), Balamb Garden (dual-faced land with card draw), Fal'Cie Paradise (conditional ramp based on opponent Islands), Tezzeret Cruel Captain (artifact-focused planeswalker), Nature's Rhythm (creature tutor with graveyard recycle), Formidable Speaker (untap tech + creature tutoring), Insidious Fungus (cheap interaction/ramp), Chord of Calling (instant-speed creature tutor), Horizon Explorer (untapped land entry + token generation). Jackson's approach: pull actual card text from [[Scryfall]], understand core mechanics, then ask what specific role each card plays in deck strategy before accepting inclusion. Distinguishes between core pieces, flex includes, and actively questionable includes.
