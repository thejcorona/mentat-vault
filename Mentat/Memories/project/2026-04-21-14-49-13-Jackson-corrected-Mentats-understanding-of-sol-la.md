# Memory

Jackson corrected Mentat's understanding of sol lands — lands that tap for 2+ mana (e.g., [[Crystal Vein]], [[Gaea's Cradle]]). Also flagged that [[Springheart Nantuko]]'s bestow ability creates a token copy of the HOST CREATURE (not Springheart itself) when landfall is triggered. When bestowed on a land-creature via [[Ashaya]] or [[Ba Sing Se]]'s Earthbend, the copy enters as a creature token but doesn't automatically chain landfall unless the original is still enchanted. The key synergy: Earthbend a sol land → sac outlet → returns tapped → landfall → Springheart pays {1}{G} to copy the enchanted creature → untapper untaps it → repeat, gaining mana each cycle from sol land's extra output.
