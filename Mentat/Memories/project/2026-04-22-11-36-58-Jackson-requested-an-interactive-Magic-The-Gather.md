# Memory

Jackson requested an interactive [[Magic: The Gathering]] game prototype where he can play against an AI opponent with card rendering from [[Scryfall]] API. [[Mentat]] built a React-based MTG arena with deck management, mana system, spell resolution, combat phases, and live card image rendering. Game includes starter decks for both player and AI, full turn structure (untap/draw/main/combat/end phases), creature combat with tapping mechanics, spell effects (Lightning Bolt, Wrath of God, Ancestral Recall, etc.), and win conditions (life total to 0). This is a substantial engineering project demonstrating Mentat's capability to build interactive game prototypes with real-time API integration.
