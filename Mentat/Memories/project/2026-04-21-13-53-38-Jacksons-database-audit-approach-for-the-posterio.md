# Memory

Jackson's database audit approach for the posterior fossa radiotherapy paper: prioritize checking LMD status (critical for baseline exclusion criterion), performance status/ECOG (inconsistently reported in literature but significant on univariate analysis), and systemic therapy era coding (highest risk for inconsistency across three cohorts spanning different treatment eras). Smoking status and race/ethnicity are normatively absent across neuro-oncology RT literature — not worth designing analysis around them, but worth flagging if <70% complete in institutional database. The referenced papers consistently omit smoking and race/ethnicity data, suggesting this is a field-wide gap rather than a study-specific problem.
