# Memory

Jackson created a drug flashcard UI component for prescribing exam prep — 50 drugs organized by category (analgesics, antibiotics, cardiovascular, respiratory, endocrine, GI, psychiatry/neurology) with flip-card mechanics. Front shows drug name, back reveals standard adult dose and clinical indication for Australian context. Includes category filtering and navigation controls. Demonstrates ability to rapidly prototype interactive study tools.
