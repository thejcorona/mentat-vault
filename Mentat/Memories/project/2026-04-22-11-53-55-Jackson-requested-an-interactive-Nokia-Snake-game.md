# Memory

Jackson requested an interactive Nokia Snake game built in [[React]]. [[Mentat]] delivered a fully functional game with Nokia 3310 aesthetic (green phosphor screen, D-pad controls, score tracking, progressive speed increase), playable via arrow keys or on-screen D-pad buttons. Game includes start/retry screens, collision detection, food spawning, and high score tracking.
